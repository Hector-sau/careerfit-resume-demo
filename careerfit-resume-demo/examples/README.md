## 每个 Demo 必须包含

- 目标岗位 JD
- 用户原始背景
- 岗位要求提取
- 匹配度分析
- 优化前简历表达
- 优化后简历表达（输出 A）
- JD 匹配建议（输出 B，不直接写入简历）
- 候选项目建议（至少 2 个方向，说明各自适合/不适合的情况）
- 信息来源与判断依据表
- 面试风险提醒
- 失败案例或注意事项
