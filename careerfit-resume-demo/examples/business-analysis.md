# 金融/商业分析简历优化 Demo

## 说明

本 Demo 用于展示"课程数据分析和活动复盘经历如何被优化成更贴近商业分析岗位的简历表达"。

参考 JD 来自实习僧发布、经国家大学生就业服务平台转载的"商业分析-战略研究实习生"岗位，招聘主体为北京三快在线科技有限公司（美团关联主体），聚焦酒店 SaaS 行业研究。本 Demo 仅用它提炼商业分析通用能力，不代表所有商业分析岗位原文。

来源链接：https://www.ncss.cn/student/jobs/GdUTJPv3FC3yCXhPRxvcUj/detail.html

## 目标岗位 JD 摘要

岗位方向：行业研究、商业分析、项目协作。

核心要求：

- 聚焦细分行业开展宏观研究，定性/定量分析、案例对标，输出行业动态报告
- 收集整理商业分析项目相关数据，提供商业洞察和建议
- 协助商业分析团队进行项目管理，保证按时交付
- 与项目协同方沟通协作，收集业务需求支持流程改进


## 简历问题诊断

原始表达的问题：

1. "做过行业分析报告"过于笼统，没有说明测算方法和数据来源。
2. 社团活动复盘的数据分析深度不够，容易被认为只是简单统计。
3. 完全没有提及数据分析工具能力（哪怕是 Excel 进阶功能），容易被认为工具能力薄弱。
4. 没有可展示的独立商业分析项目产出。

下面内容是基于 JD 的求职建议，不是当前简历内容。只有当项目真实完成并有交付物后，才可以把对应表达写进简历。

### 可补齐项目建议

项目名：细分行业商业分析报告

目标：补齐商业分析实习需要的行业研究方法、数据分析工具和结构化输出能力。

最小版本：

- 选定 1 个细分行业，完成市场规模测算
- 制作 1 份竞品对标矩阵（至少 3-5 家公司）
- 学习基础 SQL 查询语法，完成 1 个练习数据集分析
- 输出 1 份行业动态报告

项目产出物：

- 市场规模测算表
- 竞品对标矩阵
- 基础 SQL 练习记录
- 行业动态报告

完成后简历可写：

```text
独立完成细分行业商业分析报告，包含市场规模测算、竞品对标矩阵和行业动态洞察，并通过练习数据集掌握基础 SQL 查询，训练商业分析所需的数据处理与洞察输出能力。
```

### 面试风险和注意事项

- 不能把"看过行业报告"写成"独立完成行业研究"，要明确区分阅读资料和自主分析。
- 市场规模测算方法要能讲清楚数据来源和假设，不能只写结论不写过程。
- 没有真实项目数据支持时，不要编造市场规模或增长率数字。
- 这个 Demo 更适合基础商业分析/行业研究实习，不适合包装成投资分析师或战略咨询顾问经验。
