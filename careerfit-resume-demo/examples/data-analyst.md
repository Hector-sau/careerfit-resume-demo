# 数据分析岗位简历优化 Demo

## 说明

本 Demo 用于展示“普通学生经历如何被优化成更贴近数据分析岗位的简历表达”。

参考 JD 来自腾讯招聘公开岗位样例，包含企业微信数据分析师、元宝高级数据分析师、游戏数据分析等方向。部分参考岗位为正式岗，因此本 Demo 仅用于提炼数据分析岗位的通用能力，不代表实习岗位原文。

参考链接：

- http://careers.tencent.com/jobdesc.html?postId=2051914910073532416
- http://careers.tencent.com/jobdesc.html?postId=2006178595873116160
- http://careers.tencent.com/jobdesc.html?postId=2043950353740890112

## 目标岗位 JD 摘要

岗位方向：数据分析、业务分析、用户行为分析、指标体系、经营分析。

核心要求：

- 搭建和维护关键指标体系、数据看板
- 基于用户行为、留存、转化等维度分析业务问题
- 输出专题分析和经营分析报告，支持业务决策
- 协同产品、运营、研发等团队推动数据结论落地
- 熟悉 Excel、SQL、BI 工具、统计分析方法加分
- 能使用 AI 工具提升分析和报告效率加分

## 模拟用户原始背景

```text
本科经管专业，无正式数据分析实习。
课程中做过市场调研报告，用 Excel 整理问卷数据。
做过社团活动复盘，统计报名人数、到场人数和满意度。
会 Excel 基础函数，会做简单图表，不会 SQL。
想投数据分析实习，但简历里只有课程作业和社团经历。
```

## 简历问题诊断

原始表达常见问题：

1. “做过市场调研报告”没有写清数据来源、样本量、分析方法和结论。
2. “整理问卷数据”没有体现数据清洗、指标计算和图表呈现。
3. “统计活动数据”没有转化成漏斗、转化率、满意度等业务指标。
4. 缺少可展示作品，例如 Excel 看板、分析报告、SQL 练习项目。

## 输出 B：JD 匹配建议（不直接写入简历）

下面内容是基于 JD 的求职建议，不是当前简历内容。只有当项目真实完成并有交付物后，才可以把对应表达写进简历。

### 可补齐项目建议

项目名：小型用户行为数据分析作品集

目标：补齐数据分析岗位需要的指标拆解、数据清洗、图表表达和结论输出能力。

最小版本：

- 找一份公开样例数据
- 设计 5-8 个核心指标
- 用 Excel 做数据清洗和图表
- 输出 1 份分析报告
- 写入 GitHub 或作品集文档

项目产出物：

- 数据字段说明
- 指标体系表
- Excel/BI 看板截图
- 分析报告
- 简历 bullet

完成后简历可写：

```text
基于公开用户行为样例数据，搭建浏览、点击、转化等核心指标体系，使用 Excel 完成数据清洗、图表看板和分析报告输出，提出转化链路优化建议。
```

### 候选项目对比

| 候选项目 | 预计周期 | 适合选择的情况 | 不适合选择的情况 |
| --- | --- | --- | --- |

备选项目说明：用免费在线 SQL 练习平台完成 20-30 道基础查询题（筛选、聚合、分组、连表），整理成一份练习笔记，记录常见语法和踩坑点。比完整作品集轻量，适合先补技术短板。

### 面试风险和注意事项

- 不要把“整理表格”写成“搭建数据分析体系”，除非真的做了指标体系。
- 不会 SQL 可以写“正在补齐 SQL”，但不要写“熟练 SQL”。
- 没有真实业务数据时，可以用公开样例数据做作品集，但要标注数据来源。
- 数据分析简历最忌空泛，要尽量写清“数据来源、指标、方法、结论、建议”。
