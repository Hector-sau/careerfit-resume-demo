# 市场/品牌营销简历优化 Demo

## 说明

本 Demo 用于展示"内容运营和活动策划经历如何被优化成更贴近市场/品牌营销岗位的简历表达"。

参考 JD 来自腾讯招聘 Workday 页面"东欧区手游市场营销"岗位。该岗位是正式岗且带海外方向要求，本 Demo 不把它当作实习岗位原文，而是用它反推市场/品牌营销实习需要准备的能力。

来源链接：https://tencent.wd1.myworkdayjobs.com/Tencent_Careers/job/China-Shenzhen/--_R107219

## 目标岗位 JD 摘要

岗位方向：市场策略、整合营销、KOL 内容营销、本地市场洞察。

核心要求：

- 主导海外区域市场策略与中长期营销规划
- 基于产品生命周期制定整合营销方案
- 构建 KOL 与内容营销生态
- 深挖区域用户画像、审美偏好和社交媒体生态
- 协同发行、运营、研发等部门推进项目

## 模拟用户原始背景

```text
本科市场营销/传播学专业，无正式市场部实习。
运营过公开内容账号，有一定关注基础，内容主要是学习经验和生活方式，有单篇高互动内容。
做过课程市场调研报告，负责用户画像分析和竞品营销策略对比。
参与过社团活动策划，负责宣传物料设计、KOL/社群资源对接和活动复盘。
会 Excel、Canva，会用 ChatGPT 整理竞品资料和撰写文案。
想投市场/品牌营销实习，但没有真实品牌项目经验。
```

## 简历问题诊断

原始表达的问题：

1. "运营公开内容账号"没有和"内容营销""用户增长"这些市场岗位关键词建立联系。
2. 市场调研报告没有突出竞品营销策略对比这个核心能力。
3. 社团活动策划里的 KOL/资源对接经验被埋没在"活动策划"这个笼统表达里。
4. 完全没有体现对海外/区域市场的认知，容易被面试官直接判定为不了解岗位背景。

## 输出 A：优化后的简历

保守版，可以直接使用：

```text
- 独立运营公开内容账号，围绕学习经验与生活方式内容进行选题策划、标题优化和发布复盘，积累一定关注基础，有单篇高互动内容，积累内容营销与用户互动经验。
- 参与课程市场调研项目，负责目标用户画像分析和竞品营销策略对比，输出结构化调研报告。
- 参与社团活动策划与执行，负责宣传物料设计和 KOL/社群资源对接，协助完成活动传播与效果复盘。
```

增强版，需要补充项目后再使用：

```text
- 选定 1 个游戏/消费品牌，完成整合营销拆解，包含目标用户画像、内容策略、KOL 类型建议和传播节奏设计，产出营销策略文档。
```

## 输出 B：JD 匹配建议（不直接写入简历）

下面内容是基于 JD 的求职建议，不是当前简历内容。只有当项目真实完成并有交付物后，才可以把对应表达写进简历。

### 可补齐项目建议

项目名：品牌营销拆解与传播策略设计

目标：补齐市场营销实习需要的用户画像分析、内容策略设计和 KOL 合作思路。

最小版本：

- 选定 1 个游戏或消费品牌，梳理其目标用户画像
- 分析 3-5 个同类品牌的内容营销打法
- 设计 1 份传播节奏和 KOL 类型建议

项目产出物：

- 用户画像文档
- 竞品内容营销对比表
- 传播策略文档

完成后简历可写：

```text
针对 1 个游戏/消费品牌完成整合营销拆解，包含目标用户画像分析、竞品内容营销对比和 KOL 类型建议，产出传播策略文档，训练市场营销所需的用户洞察与内容策略设计能力。
```

### 面试风险和注意事项

- 不能把"发过公开内容平台"写成"具备内容营销策略能力"，要说明具体的选题、数据和复盘逻辑。
- 不了解海外市场时，不要在简历里暗示自己有海外项目经验。
- 没有真实传播数据时，不要编造曝光量或转化率。
- 这个 Demo 更适合基础市场营销/品牌运营/内容营销实习，不适合包装成海外发行或成熟品牌经理经验。
